## Summary

Milestone v1.1 made the Host/WSL workspace foundation visible, configurable, safer, and resilient. It added contextual workspace selection, unified backend routing, durable trusted profiles, safe per-chat switching, approval and process guards, settings migration, and compatibility management.

## Key Accomplishments

1. Delivered contextual Host/WSL workspace selection with explicit status, validation, and no-fallback execution.
2. Unified commands, filesystem access, discovery, and skills behind one preflighted workspace backend.
3. Added durable searchable workspace profiles with trust, preference, lifecycle management, and safe per-chat switching.
4. Added invocation locks, scoped approvals, destructive confirmation, repository identity checks, and structured recovery errors.
5. Added versioned settings migration, malformed-data recovery, downgrade/non-Windows safety, and bounded redacted approval history.
6. Added isolated per-chat workspace restoration and capability-gated session resume without transcript copying.
