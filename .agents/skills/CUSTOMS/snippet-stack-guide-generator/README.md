# Snippet Stack Guide Generator

Creates clear user guides for prompt/snippet libraries.

Use it when a snippet library has multiple entry points, modes, aliases, or execution order rules and users need a practical “which snippet do I use?” guide.

## Files

- `SKILL.md` - activation instructions
- `references/stack-analysis.md` - extraction procedure
- `references/guide-patterns.md` - documentation and diagram patterns
- `templates/snippet-stack-guide-template.md` - repo-ready markdown template

## Typical Outputs

- snippet inventory
- start-here decision table
- intended user flows
- Mermaid diagrams
- invocation examples
- stop and continuation rules
- compatibility notes
