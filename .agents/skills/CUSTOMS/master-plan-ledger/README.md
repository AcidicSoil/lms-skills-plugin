# master-plan-ledger

Standalone skill for converting one or more planning artifacts into a deterministic Master Plan and Merge Ledger.

## Trigger Examples

- "Merge these planning docs into a master plan."
- "Create a master plan and merge ledger from these PRDs."
- "Update this prior master plan with the latest notes."
- "Preserve plan conflicts and show convergence actions."

## Output Blocks

1. `<<<MASTER_PLAN v1>>>`
2. `<<<MERGE_LEDGER v1>>>`
3. `<<<NEXT_ITERATION_FOCUS v1>>>`
